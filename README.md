# Сравнительный анализ моделей детекции (COCO 2017)

Автоматизированный пайплайн для проведения инференса и оценки метрик (mAP, AR, Speed) моделей детекции объектов (LibreYOLO9, LibreYOLOX, LibreDEIM) в формате ONNX.

## Быстрый старт

Выполните следующие шаги для развертывания окружения и запуска бенчмарка:

### 1. Клонирование репозитория
```bash
git clone <URL_ВАШЕГО_РЕПОЗИТОРИЯ>
cd <ИМЯ_ПАПКИ_РЕПОЗИТОРИЯ>
```

### 2. Создание и активация виртуального окружения
```bash
python3 -m venv venv
source venv/bin/activate
```

### 3. Установка зависимостей
```bash
pip install -r requirements.txt
```

### 4. Скачивание весов моделей (ONNX)
Автоматический скрипт скачает все необходимые модели с Hugging Face и разложит их по папкам внутри директории `weights/`:
```bash
python3 download_weights.py
```

### 5. Подготовка датасета COCO
Поместите валидационную выборку COCO2017 в корень проекта так, чтобы структура директорий выглядела следующим образом:
```text
datasets/
└── coco/
    ├── annotations/
    │   └── instances_val2017.json
    └── val2017/
        ├── 000000000139.jpg
        └── ...
```

### 6. Запуск пайплайна
Запустите главный скрипт. Он автоматически проведет инференс для всех моделей, посчитает метрики и сгенерирует отчет:
```bash
python3 run_pipeline.py
```

---

## Результаты работы

После завершения работы скрипта `run_pipeline.py` в корне проекта автоматически появятся:
* `Benchmark_Report.md` — итоговый отчет со сводной таблицей метрик (mAP50-95, mAP50, AR и т.д.) и временем кадра (ms).
* Папка `predictions/` — сохраненные результаты предсказаний в формате `.json` для каждой модели.
* Папка `images/` — сгенерированные графики **Precision-Recall Curve** для каждой модели.
